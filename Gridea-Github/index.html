Gridea GitHub 博客

2020-04-15

Gridea是一个很好用的博客创建、编写的平台

Gridea支持Markdown语法和Katex公式渲染，你可以向世界展示你的博客。

1. 准备

准备好Gridea和Github账户

注： 在这里，我将以Linux版的Gridea来示范。

2. 基本配置

点击Github首面右上角的+，选择new repository。

Name填写用户名.github.io
其它的只要选择Public即可

进入Github>>Setting>>Developer settings>>Personal access tokens并注册一个Token

  1. 点击右侧的Generate new token
  2. 在Note处填写你的blog名称
  3. 勾选reop
  4. 点击绿色的创建按钮

进入Github>>Setting>>Developer settings>>Oauth Apps

  1. 创建一个新的Oauth App
  2. Application name 填写blog名称
  3. Homepage URL 填写blog地址即用户名.github.io
  4. Application description 随意填写
  5. Authorization callback URL 填写blog地址
  6. 将Client ID和Client Secret记录下来

打开Gridea

找到左侧的远程模块

  1. 域名填写blog地址
  2. 仓库名称填写用户名.github.io
  3. 仓库用户名就是你的用户名
  4. 分支填写master即可
  5. 邮箱就是你GitHub注册用的邮箱
  6. 令牌填写Github的密码
  7. CNAME填写你的网址即可

点击保存按钮

再点击检测远程连接按钮

3. 评论配置

进入Gridea>>远程>>评论。

  1. 选择Gitalk
  2. 填入所有信息

点击保存按钮

4. 上传

点击左侧的上传按钮

等侯2分钟

进入你的网页，就能看到了
