Timer

2020-04-13

Timer是由LanGong开发组开发的桌面时钟应用程序

使用方法

您必须先安装git这个软件包。

1. 下载

在终端命令行内输入以下命令

git clone https://github.com/langonginc/Timer.git

2. 使用

在终端命令行内输入以下命令

bash ~/Timer/timer.sh

进入程序后，按照提示输入设定秒数并等待。

3. Debug

Error-01：您没有输入数字

Error-02：您输入的不是有效数字（在某些版本中可能不会报错）

Error-254：未知错误

Error-Signed：运行时错误
