Timer

2020-04-13

Timer 是由 LanGong 开发组开发的桌面时钟应用程序

使用方法

您必须先安装zip和unzip这两个软件包。

1. 下载

进入这个左侧的下载页面

选择Timer并下载

下载后在终端命令行中输入以下命令：

# 1. **解压刚刚下载的zip压缩包并进入文件夹
unzip Timer.zip
cd Timer

# 2. 安装Timer
bash install.sh

2. 使用

在终端命令行内输入以下命令：

bash ~/timer.sh

进入程序后，按照提示输入设定秒数并等待。

3. Debug

Error-01：您没有输入数字

Error-02：您输入的不是有效数字（在某些版本中可能不会报错）

Error-254：未知错误
